# Movie Recommendation System - Complete Requirements Document

## PROJECT OVERVIEW

Build a movie recommendation web application for a college project with React frontend and Python Flask backend that uses BOTH watch history and ratings to generate personalized recommendations.

### Core Functionality
Users can:
1. Browse and search movies from a provided dataset
2. Click "Watch" on movies (simulated - records watch event with metadata)
3. Rate movies (1-5 stars) - optional but encouraged after watching
4. Get personalized recommendations based on watch history + ratings using content-based filtering

### Why This Approach?
This system combines **IMPLICIT feedback** (what users watch) and **EXPLICIT feedback** (how users rate) for better accuracy - mimicking industry standards like Netflix and YouTube.

## RECOMMENDATION ALGORITHM - DETAILED SPECIFICATION

### Algorithm Type
**Enhanced Multi-Factor Content-Based Filtering**

### Scoring Components
1. **Genre Matching** - 40% weight
2. **Director Matching** - 25% weight
3. **Actor Matching** - 20% weight
4. **Quality Score (IMDB Rating)** - 15% weight

---

### Step 1: Calculate Interaction Scores

Determine how much each user interaction should influence recommendations:

```python
def calculate_interaction_score(movie, user_id):
    """
    Assigns weight to user-movie interaction
    Returns: float between -0.5 and 1.0
    """
    watched = check_if_watched(user_id, movie)
    rating = get_rating(user_id, movie)
    
    # Combined interactions (strongest signals)
    if watched and rating:
        if rating == 5:
            return 1.0      # Strongest positive
        elif rating == 4:
            return 0.8      # Strong positive
        elif rating == 3:
            return 0.5      # Neutral/mild positive
        elif rating == 2:
            return -0.3     # Mild negative (avoid)
        else:  # rating == 1
            return -0.5     # Strong negative (definitely avoid)
    
    # Implicit feedback only
    elif watched and not rating:
        return 0.5          # Moderate positive signal
    
    # Explicit feedback only
    elif rating and not watched:
        if rating >= 4:
            return 0.6      # Positive signal
        elif rating == 3:
            return 0.3      # Mild positive
        else:
            return -0.3     # Negative signal
    
    return 0  # No interaction
```

**Interaction Weight Table**:
| Interaction Type | Weight | Interpretation |
|-----------------|--------|----------------|
| Watched + Rated 5â˜… | 1.0 | Loved it |
| Watched + Rated 4â˜… | 0.8 | Liked it |
| Watched + Rated 3â˜… | 0.5 | Okay |
| Watched only | 0.5 | Interested |
| Rated 5â˜… only | 0.6 | Knows & loves |
| Watched + Rated 2â˜… | -0.3 | Didn't like |
| Watched + Rated 1â˜… | -0.5 | Hated it |

### Step 2: Build User Preference Profile

Extract weighted preferences from all positive interactions:

```python
def build_user_profile(user_id):
    """
    Creates comprehensive user preference profile
    Returns: dict with weighted genre/director/actor preferences
    """
    # Get all user interactions
    watched_movies = get_watch_history(user_id)
    ratings = get_user_ratings(user_id)
    
    # Initialize weighted preferences
    genre_weights = {}
    director_weights = {}
    actor_weights = {}
    
    # Process each interaction
    for movie in watched_movies:
        interaction_score = calculate_interaction_score(movie, user_id)
        
        # Only consider positive interactions
        if interaction_score > 0:
            # Weight genres
            for genre in movie.genres:
                genre_weights[genre] = genre_weights.get(genre, 0) + interaction_score
            
            # Weight director
            director_weights[movie.director] = director_weights.get(movie.director, 0) + interaction_score
            
            # Weight actors (all 4 stars)
            for actor in [movie.star1, movie.star2, movie.star3, movie.star4]:
                if actor:
                    actor_weights[actor] = actor_weights.get(actor, 0) + interaction_score
    
    return {
        'genres': genre_weights,
        'directors': director_weights,
        'actors': actor_weights,
        'total_positive_interactions': len([m for m in watched_movies 
                                           if calculate_interaction_score(m, user_id) > 0])
    }
```

### Step 3: Score Candidate Movies

Calculate recommendation score for each unwatched movie:

```python
def score_candidate_movie(movie, user_profile):
    """
    Calculate how well movie matches user preferences
    Returns: dict with final_score and breakdown
    """
    
    # 1. GENRE MATCHING (40% weight)
    genre_score = 0
    total_genre_weight = sum(user_profile['genres'].values())
    
    if total_genre_weight > 0:
        for genre in movie.genres:
            if genre in user_profile['genres']:
                genre_score += user_profile['genres'][genre]
        genre_score = genre_score / total_genre_weight
    
    # 2. DIRECTOR MATCHING (25% weight)
    director_score = 0
    total_director_weight = sum(user_profile['directors'].values())
    
    if total_director_weight > 0 and movie.director in user_profile['directors']:
        director_score = user_profile['directors'][movie.director] / total_director_weight
    
    # 3. ACTOR MATCHING (20% weight)
    actor_score = 0
    total_actor_weight = sum(user_profile['actors'].values())
    
    if total_actor_weight > 0:
        movie_actors = [movie.star1, movie.star2, movie.star3, movie.star4]
        for actor in movie_actors:
            if actor and actor in user_profile['actors']:
                actor_score += user_profile['actors'][actor]
        actor_score = actor_score / (total_actor_weight * 4)
    
    # 4. QUALITY SCORE (15% weight)
    quality_score = movie.imdb_rating / 10.0
    
    # FINAL WEIGHTED SCORE
    final_score = (
        genre_score * 0.40 +
        director_score * 0.25 +
        actor_score * 0.20 +
        quality_score * 0.15
    )
    
    return {
        'movie': movie,
        'final_score': final_score,
        'breakdown': {
            'genre_score': genre_score,
            'director_score': director_score,
            'actor_score': actor_score,
            'quality_score': quality_score
        }
    }
```

**Scoring Formula**:
```
Final Score = (Genre Match Ã— 0.40) + (Director Match Ã— 0.25) + 
              (Actor Match Ã— 0.20) + (IMDB Quality Ã— 0.15)

Where each component score ranges from 0.0 to 1.0
```

### Step 4: Generate Recommendations with Explanations

```python
def get_recommendations(user_id, num_recommendations=12):
    """
    Main recommendation function
    Returns: dict with recommendations and explanations
    """
    # Build user profile
    user_profile = build_user_profile(user_id)
    
    # Check minimum threshold
    if user_profile['total_positive_interactions'] < 3:
        return {
            'success': False,
            'message': 'Watch or rate at least 3 movies to get recommendations',
            'current_count': user_profile['total_positive_interactions'],
            'required_count': 3
        }
    
    # Get all movies
    all_movies = load_movies()
    
    # Get watched movie IDs
    watched_ids = get_watched_movie_ids(user_id)
    
    # Filter to unwatched movies only
    candidate_movies = [m for m in all_movies if m.id not in watched_ids]
    
    # Score each candidate
    scored_movies = []
    for movie in candidate_movies:
        result = score_candidate_movie(movie, user_profile)
        scored_movies.append(result)
    
    # Sort by score (descending)
    scored_movies.sort(key=lambda x: x['final_score'], reverse=True)
    
    # Generate human-readable explanations
    recommendations = []
    for item in scored_movies[:num_recommendations]:
        movie = item['movie']
        breakdown = item['breakdown']
        
        # Build explanation based on highest contributing factors
        reasons = []
        
        if breakdown['genre_score'] > 0.5:
            top_genres = get_matching_genres(movie, user_profile['genres'])
            reasons.append(f"You love {', '.join(top_genres[:2])} movies")
        
        if breakdown['director_score'] > 0.5:
            reasons.append(f"You enjoyed other films by {movie.director}")
        
        if breakdown['actor_score'] > 0.3:
            top_actors = get_matching_actors(movie, user_profile['actors'])
            reasons.append(f"Stars {top_actors[0]} from your favorites")
        
        if breakdown['quality_score'] > 0.8:
            reasons.append(f"Highly rated ({movie.imdb_rating}/10)")
        
        recommendations.append({
            'movie': movie,
            'score': item['final_score'],
            'reasons': reasons if reasons else ['Based on your viewing history']
        })
    
    return {
        'success': True,
        'recommendations': recommendations,
        'based_on': {
            'total_watched': len(watched_ids),
            'total_rated': get_rating_count(user_id),
            'total_interactions': len(watched_ids)
        }
    }
```

## EDGE CASES TO HANDLE

### 1. Insufficient Data
```
User has < 3 interactions
â†’ Display: "Watch or rate at least 3 movies to get recommendations"
â†’ Show progress: "2/3 interactions"
â†’ Disable "Get Recommendations" button
```

### 2. All Movies Watched
```

â†’ Display: "You've watched everything! Check back for new additions."
â†’ Show: "Your Favorites" section with highest-rated movies
```

### 3. No Positive Interactions
```
User watched 5 movies, all rated 1-2 stars
â†’ Cannot build positive preference profile
â†’ Display: Popular movies (highest IMDB ratings)
â†’ Message: "Try rating some movies 4-5 stars for better recommendations"
```

### 4. Tie Scores
```
Two movies have identical recommendation score
â†’ Use IMDB rating as tiebreaker (higher first)
â†’ If still tied, use release year (newer first)
```

### 5. Empty Search Results
```
User searches for non-existent movie
â†’ Display: "No movies found matching '[search term]'"
â†’ Show "Clear search" button
â†’ Suggest: "Try different keywords"
```

### 6. API Failure
```
Backend unavailable or request fails
â†’ Display: "Unable to load recommendations. Please try again."
â†’ Show retry button
â†’ Don't crash the app
â†’ Log error for debugging
```

### 7. No Matching Genre Filter
```
User selects multiple incompatible genres
â†’ Display: "No movies found with these genres"
â†’ Show selected filters
â†’ Suggest: "Try fewer filters"
```

### 8. Conflicting Preferences
```
User likes both Horror and Romance (rare combo)
â†’ Algorithm should handle gracefully
â†’ Find best matches or diversify recommendations
â†’ Weight by individual genres if no combo exists
```
